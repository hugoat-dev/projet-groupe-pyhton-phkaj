# projet-groupe-pyhton-phkaj
Eco-Hero (Projet TI250 EFREI) : Jeu éducatif Python pour enfants (6-10 ans). Explorez divers biomes (Océans, Forêts, Espace) et collectez des déchets via une trajectoire linéaire. Un projet Earth Tech alliant sensibilisation écologique et éco-conception pour réduire l'empreinte carbone numérique.
